Strategic Performance Analysis for Royal Challengers Bangalore (RCB)


Description: Acted as a Sports Data Analyst to develop a data-driven strategy for the RCB franchise ahead of the 2017 mega auction. The goal was to optimize player recruitment and match-day tactics by analyzing historical IPL data (ball-by-ball, player stats, and venue metrics) using complex SQL queries.

Key Insights & Contributions:


1.Auction Strategy: Identified high-impact all-rounders (e.g., Watson and Jadeja) and consistent match-winners (V. Kohli, AB de Villiers) by calculating advanced metrics like Batting Average per Season and Wickets per Venue.


2.Venue-Specific Performance: Discovered that RCB maintains a significantly higher win rate at M. Chinnaswamy (55.17%) compared to away grounds, leading to a recommendation for venue-based squad rotation.


3.Tactical Decision Making: Analyzed the correlation between toss decisions and match outcomes, identifying specific stadiums (e.g., MA Chidambaram) where batting first increased win probability by over 70%.


4.Data Engineering: Developed a robust relational database schema involving multiple tables (Matches, Ball_by_Ball, Players, Venues) and performed extensive data cleaning to fix naming inconsistencies.


5.Performance Benchmarking: Used SQL to rank players by "Value for Money," focusing on death-over bowling economy and middle-order strike rates to address historical "run leakage" issues.


Tools Used: MySQL (CTEs, Window Functions, Joins, Subqueries), Information Schema, MS Excel, PowerPoint.
